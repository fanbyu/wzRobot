# 电机驱动模块快速测试指南

## 🚀 快速开始（3步完成）

### 步骤1：运行安装脚本

在项目目录中打开PowerShell，执行：

```powershell
.\安装修复版电机驱动.ps1
```

这个脚本会自动：
- ✅ 删除旧扩展
- ✅ 安装V0.4.1新版本
- ✅ 复制依赖库到Mind+全局目录
- ✅ 清理Mind+缓存

### 步骤2：重启Mind+

**完全关闭** Mind+，然后重新打开。

> ⚠️ **重要**：必须完全关闭，不能只是最小化！

### 步骤3：加载测试程序

在Mind+中：
1. 点击"文件" → "打开"
2. 选择 `examples/电机驱动板测试.mp`
3. 连接Arduino开发板
4. 点击"上传"按钮

## 🔍 验证方法

### 方法1：观察硬件反应

上传成功后，观察连接的电机：

| 电机类型 | 预期行为 |
|---------|---------|
| DC电机 | 正转1秒 → 停止1秒 → 循环 |
| 编码器电机 | 正转1秒 → 停止1秒 → 循环 |
| 步进电机 | 按指定步数转动 |
| 舵机 | 从0°转到180° → 回到0° → 循环 |

### 方法2：查看生成的代码

在Mind+中切换到"代码"视图，检查生成的C++代码：

#### ✅ 正确的代码应该是：

```cpp
#include <Emakefun_MotorDriver.h>

// 创建对象
Emakefun_MotorDriver mMotorDriver(0x60);
Emakefun_DCMotor* DCmotor_1;

void setup() {
    Serial.begin(9600);
    mMotorDriver.begin(1000);
    DCmotor_1 = mMotorDriver.getMotor(1);
}

void loop() {
    DCmotor_1->setSpeed(200);
    DCmotor_1->run(FORWARD);
    delay(1000);
    DCmotor_1->run(RELEASE);
    delay(1000);
}
```

#### ❌ 如果看到这样的代码，说明安装有问题：

```cpp
// 错误示例1：对象赋值而非初始化
Emakefun_MotorDriver mMotorDriver;
void setup() {
    mMotorDriver = Emakefun_MotorDriver(0x60);  // ❌
}

// 错误示例2：非指针类型
Emakefun_DCMotor DCmotor_1;
void setup() {
    *DCmotor_1 = mMotorDriver.getMotor(1);  // ❌
}

// 错误示例3：重复声明
void loop() {
    Emakefun_DCMotor* DCmotor_1;  // ❌ 每次都声明
    DCmotor_1 = mMotorDriver.getMotor(1);
}
```

## ⚠️ 常见问题排查

### Q1: 编译错误：'DCmotor_1' was not declared in this scope

**原因：** 忘记添加"DC电机 [1] 初始化"积木

**解决：** 在"当启动时"中添加对应的初始化积木

---

### Q2: 编译错误：redefinition of 'DCmotor_1'

**原因：** 扩展未正确更新，仍使用旧版本

**解决：** 
1. 完全关闭Mind+
2. 重新运行 `.\安装修复版电机驱动.ps1`
3. 清理缓存目录
4. 重新打开Mind+

---

### Q3: 编译成功但电机不动

**可能原因：**
1. 接线错误
2. 电源不足
3. I2C地址不正确

**排查步骤：**
1. 检查电机驱动板I2C地址是否为0x60
2. 检查电机电源是否正常（通常需要外部电源）
3. 检查电机接线是否正确
4. 尝试更换电机测试

---

### Q4: 找不到扩展

**原因：** 扩展未正确安装

**解决：**
```powershell
# 检查扩展是否存在
Test-Path "$env:LOCALAPPDATA\DFScratch\extensions\fan-wzrobot-thirdex"

# 如果不存在，重新安装
.\安装修复版电机驱动.ps1
```

---

### Q5: Mind+中看不到新的积木

**原因：** 缓存未清理

**解决：**
```powershell
# 手动清理缓存
Remove-Item "$env:LOCALAPPDATA\DFScratch\cache\*" -Force
Remove-Item "$env:LOCALAPPDATA\DFScratch\build\*" -Recurse -Force

# 完全重启Mind+
```
