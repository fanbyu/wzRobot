## 🎯 主要更新

### 核心修复：电机驱动模块

本次更新主要解决了电机驱动模块的关键bug，使所有类型的电机都能正常工作。

#### 修复内容

1. **✅ 修正对象初始化语法**
   - 从赋值语法改为构造函数初始化
   - `mMotorDriver(0x60)` 替代 `mMotorDriver = Emakefun_MotorDriver(0x60);`

2. **✅ 修正指针类型声明**
   - 所有电机对象改为正确的指针类型
   - `Emakefun_DCMotor*`、`Emakefun_EncoderMotor*`、`Emakefun_StepperMotor*`、`Emakefun_Servo*`

3. **✅ 移除重复变量声明**
   - 控制函数中不再重复调用`addObject`
   - 避免编译错误：redefinition of variable

4. **✅ 完善代码生成逻辑**
   - 17个函数得到修复
   - 生成的C++代码符合标准规范

### 关键代码变更

#### 1. 电机驱动初始化
```typescript
// Before (V0.3.9)
Generator.addObject('mMotorDriver', 'Emakefun_MotorDriver', 
    'mMotorDriver = Emakefun_MotorDriver(0x60);', true);

// After (V0.4.1)
Generator.addObject('mMotorDriver', 'Emakefun_MotorDriver', 
    'mMotorDriver(0x60)', true);
```

#### 2. DC电机对象获取
```typescript
// Before (V0.3.9)
Generator.addObject(`DCmotor_1`, 'Emakefun_DCMotor', 
    `*DCmotor_1 = mMotorDriver.getMotor(1);`, true);

// After (V0.4.1)
Generator.addObject(`DCmotor_1`, 'Emakefun_DCMotor*', 
    `DCmotor_1 = mMotorDriver.getMotor(1)`, true);
```

#### 3. 控制函数优化
```typescript
// Before (V0.3.9) - 每次调用都重复声明
export function controlEncoderMotor(parameter: any) {
    Generator.addObject(`EncodeMotor_1`, 'Emakefun_EncoderMotor', 
        `*EncodeMotor_1 = mMotorDriver.getEncoderMotor(1);`);
    Generator.addCode(`EncodeMotor_1->setSpeed(${speed});`);
}

// After (V0.4.1) - 直接使用已声明的变量
export function controlEncoderMotor(parameter: any) {
    Generator.addCode(`EncodeMotor_1->setSpeed(${speed});`);
}
```

## 🚀 升级指南

### 从 V0.3.9 升级到 V0.4.1

#### 方法1：使用自动安装脚本（推荐）

```powershell
# 在项目目录中执行
.\安装修复版电机驱动.ps1
```

脚本会自动完成：
1. 删除旧版本扩展
2. 安装V0.4.1新版本
3. 复制依赖库到Mind+全局目录
4. 清理Mind+缓存

#### 方法2：手动安装

```powershell
# 1. 删除旧扩展
Remove-Item "$env:LOCALAPPDATA\DFScratch\extensions\fan-wzrobot-thirdex" -Recurse -Force

# 2. 解压新扩展
Add-Type -AssemblyName System.IO.Compression.FileSystem
[System.IO.Compression.ZipFile]::ExtractToDirectory(
    "wzRobot-V0.4.1.mpext", 
    "$env:LOCALAPPDATA\DFScratch\extensions\fan-wzrobot-thirdex"
)

# 3. 清理缓存
Remove-Item "$env:LOCALAPPDATA\DFScratch\cache\*" -Force
Remove-Item "$env:LOCALAPPDATA\DFScratch\build\*" -Recurse -Force

# 4. 复制依赖库
Copy-Item "arduinoC\libraries\Emakefun_MotorDriver\*" `
    "C:\Program Files (x86)\Mind+\Arduino\libraries\Emakefun_MotorDriver\" `
    -Recurse -Force
```

### 验证升级

1. **完全关闭并重新打开Mind+**
2. 加载测试程序：`examples/电机驱动板测试.mp`
3. 上传到Arduino开发板
4. 观察电机是否正常工作
