## 🛠️ 工具脚本

### 1. build.py

**用途**: 打包 Mind+ 扩展  
**用法**: `python build.py`  
**输出**: `wzRobot-V*.mpext`

---

### 2. check_portability.py

**用途**: 检查库文件完整性和可移植性  
**用法**: `python check_portability.py`  
**输出**: 检查报告 + `可移植性检查报告.md`

---

### 3. generate_precompiled.py

**用途**: 生成 Arduino 库的预编译 .o 文件  
**用法**: `python generate_precompiled.py`  
**输出**: 所有必需的 .o 文件

## 📋 文档使用流程

### 新手入门流程

```
1. 阅读 README.md          → 了解项目
2. 阅读 项目使用指南.md     → 学习基本用法
3. 阅读 避坑指南.md         → 了解常见陷阱
4. 开始开发...
```

### 开发工作流程

```
1. 修改代码
2. 运行 check_portability.py   → 检查完整性
3. 运行 generate_precompiled.py → 生成预编译文件
4. 运行 build.py                → 打包扩展
5. 在干净环境测试
6. 查阅 完全指南.md             → 遇到问题时
7. 发布
```

### 故障排查流程

```
1. 查看错误信息
2. 查阅 完全指南.md → "常见问题解决"章节
3. 查阅 避坑指南.md  → 搜索相关关键词
4. 运行检查脚本
5. 清理缓存并重试
```
