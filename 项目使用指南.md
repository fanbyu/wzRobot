## 🎯 项目简介

这是一个为Mind+编程环境开发的Arduino第三方扩展库，集成了以下硬件模块：

1. **五路循迹传感器** - 红外循线检测
2. **电机驱动板** - DC电机、编码器电机、步进电机、舵机控制
3. **Sentry2视觉传感器** - 颜色/线条/Blob/卡片/二维码/人脸识别
4. **RGB超声波模块** - RGB LED灯效 + 超声波测距

## 🚀 快速开始

### 方式一：直接使用已编译的扩展包（推荐）

如果您只是想使用这个扩展，不需要修改源码：

1. **运行安装脚本**：
   ```powershell
   .\安装扩展.ps1
   ```

2. **或者手动安装**：
   - 将 `wzRobot-V0.3.8.mpext` 复制到 Mind+ 扩展目录
   - 路径：`%LOCALAPPDATA%\DFScratch\extensions\`
   - 解压到 `fan-wzrobot-thirdex` 文件夹

3. **重启Mind+**即可使用

### 方式二：修改源码后重新编译

如果您需要修改或定制扩展功能：

#### 步骤1：安装依赖

确保您的系统已安装：
- Python 3.x
- Mind+ 编程环境

#### 步骤2：修改源码

主要修改文件：
- `arduinoC/main.ts` - 积木定义和代码生成逻辑
- `arduinoC/libraries/` - 底层驱动库
- `config.json` - 扩展配置

#### 步骤3：编译打包

```bash
python build.py
```

这将生成新的 `wzRobot-V0.3.8.mpext` 文件。

#### 步骤4：安装依赖库到全局

```powershell
# 复制电机驱动库
Copy-Item "arduinoC\libraries\motordriverboard\*" "C:\Program Files (x86)\Mind+\Arduino\libraries\Emakefun_MotorDriver\" -Recurse -Force

# 复制Sentry2库
Copy-Item "arduinoC\libraries\Sentry-Arduino" "C:\Program Files (x86)\Mind+\Arduino\libraries\Sentry-Arduino" -Recurse -Force
```

#### 步骤5：清理缓存并安装

```powershell
# 清理Mind+缓存
.\清理缓存.ps1

# 安装新扩展
.\安装扩展.ps1
```

#### 步骤6：重启Mind+测试

## ⚠️ 注意事项

### 1. 缓存问题
每次修改库文件后，**必须**清理Mind+缓存：
```powershell
Remove-Item "$env:LOCALAPPDATA\DFScratch\cache\*" -Force
Remove-Item "$env:LOCALAPPDATA\DFScratch\build\*" -Recurse -Force
```
然后完全关闭并重新打开Mind+。

### 2. 库文件位置
Mind+编译时需要从全局Arduino库目录加载库文件，确保库文件已复制到：
```
C:\Program Files (x86)\Mind+\Arduino\libraries\
```

### 3. 预编译文件
如果遇到链接错误，可能需要生成预编译的 `.o` 文件。详见《Mind+扩展库编译与链接问题排查备忘.md》。

### 4. 版本兼容性
- 当前版本：V0.3.6
- 最新编译版本：V0.3.8
- 建议使用最新的V0.3.8版本

## 🆘 常见问题

### Q: 编译时找不到头文件？
**A**: 确保库文件已复制到全局Arduino库目录。

### Q: 链接错误？
**A**: 检查是否生成了预编译的 `.o` 文件，参考详细文档。

### Q: 积木面板拥挤？
**A**: V0.3.6已修复此问题，使用了分类标签进行视觉分隔。

### Q: 代码无法生成？
**A**: V0.3.6已修复，所有 `Blockly.Arduino.ORDER_ATOMIC` 已替换为 `Generator.ORDER_ATOMIC`。
